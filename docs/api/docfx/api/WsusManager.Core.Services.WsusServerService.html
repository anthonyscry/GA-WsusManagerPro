    Class WsusServerService

    Connects to the local WSUS server via the Microsoft.UpdateServices.Administration managed API loaded at runtime. All WSUS API calls are blocking, so they are wrapped in Task.Run to keep the UI responsive.

    ApproveUpdatesAsync(int, IProgress<string>?, CancellationToken)

    Auto-approves pending updates matching approved classifications against the "All Computers" target group. Approved classifications: Critical Updates, Security Updates, Update Rollups, Service Packs, Updates, Definition Updates. Upgrades are never auto-approved. Preview and Beta titles are excluded. Safety threshold blocks approval if count exceeds maxCount. Returns the count of updates approved.
