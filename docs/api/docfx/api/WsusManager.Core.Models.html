    Namespace WsusManager.Core.Models

    Classes

    AppSettings

    Application settings model. Serialized to/from JSON at %APPDATA%\WsusManager\settings.json. Survives close/reopen cycles.

    ClientDiagnosticResult

    Snapshot of WSUS client configuration and state on a remote host, gathered from the Windows Update registry hive, service control manager, and WMI.

    ConnectivityTestResult

    Result of a network connectivity test from the client host to the WSUS server. Ports are tested with a TCP connection attempt; latency is measured via ping.

    DashboardData

    Holds all data collected during a dashboard refresh cycle. Populated by IDashboardService.CollectAsync().

    DiagnosticCheckResult

    Result of a single health check. Each check has a name, status, detail message, and optional repair outcome if auto-repair was attempted.

    DiagnosticReport

    Aggregate result of a full diagnostics run. Contains all individual check results and computed summary statistics. Immutable â€” created once after all checks complete.

    ExportOptions

    ImportOptions

    InstallOptions

    Options for the WSUS + SQL Server Express installation wizard. All fields are collected via the GUI dialog before installation begins.

    OperationResult

    Represents the result of a service operation. Every public service method returns this type instead of throwing exceptions for expected failures. Exceptions are reserved for programming errors only.

    OperationResult<T>

    Generic version of OperationResult that includes a data payload.

    ProcessResult

    Result of an external process execution.

    ScheduledTaskOptions

    Options for creating a Windows scheduled task for WSUS maintenance. All fields are collected via the ScheduleTaskDialog before task creation.

    ServiceStatusInfo

    Status snapshot of a Windows service at a point in time.

    SyncResult

    Result of a WSUS synchronization operation.

    WsusErrorCodes

    Static dictionary of common WSUS and Windows Update error codes with descriptions and recommended fixes. Used by the error-code lookup feature (CLI-06).

    Keys are uppercase hex codes without the "0x" prefix (e.g., "80244010"). The Lookup(string?) method accepts either format.

    WsusErrorInfo

    Decoded information for a WSUS or Windows Update error code. Sourced from the WsusErrorCodes static dictionary.
