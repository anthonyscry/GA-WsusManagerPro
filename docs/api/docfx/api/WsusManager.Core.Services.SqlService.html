    Class SqlService

    Centralized SQL execution service for WSUS database operations. All connection strings use Integrated Security and TrustServerCertificate. Operations with commandTimeoutSeconds = 0 run with unlimited timeout — required for maintenance queries (index rebuild, shrink, batch deletes).
