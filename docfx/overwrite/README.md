# WsusManager.Core Namespace

Core library containing business logic, services, and models for WSUS operations.

## Key Components

- **Services**: Business logic interfaces and implementations for WSUS management operations
- **Models**: Data transfer objects and entities representing WSUS resources
- **Infrastructure**: Logging, process execution, admin checks, and path validation

This library has no UI dependencies and can be used independently for testing or automation.
